# Undergraduate Bioinformatics & Data Analysis Projects

Welcome to my portfolio! I am Velda, a final-year BBiomedSc (biomedical science) student at Monash University. I am interested in pursuing further studies in **bioinformatics**. This repository contains a selection of introductory Python and R projects I have completed through university coursework and self-directed learning. 

## 🧬 Python Projects

### 1. DNA Motif Finder
* Language: Python
* Context: Completed as part of a Coursera Bioinformatics specialization.
* Description: A computational script designed to identify recurring DNA motifs within biological sequences. This project helped me understand string manipulation, algorithmic efficiency, and basic genomic data processing.

## 📊 R Projects

### 1. Clinical Trial Data Management 
* Language: R
* Context: Final project for PBH3010 [Health Data Management].
* Description: The ClearHead study is a mock clinical trial for a migraine medication. This study was generated as a course material only. This project helped me learn how to clean and merge raw clinical trial datasets, handle missing values, and generate summary statistics for patient demographics.

### 2. Physiological Data Visualization
* Language: R
* Context: Independent side-project using class-sourced physiological data.
* Description: An R script utilizing `ggplot2` to visualize urinary flow rate for different drinks consumed. 
* Note: The raw dataset has been excluded from this repository to maintain the privacy and ethical handling of my classmates' physiological data. Only the visualization script is provided.

## 🛠️ Skills Demonstrated
* Languages: Python, R
* Concepts: Data cleaning, data visualisation, basic sequence analysis.
